# 大孚数据可视化视觉系统 - 详细设计文档

## 1. 项目概述
本项目旨在构建一个高性能、高视觉保真度的数据可视化大屏（Data Visualization Dashboard）。系统主要用于展示关键业务指标、服务器状态、网络安全态势及机构分布情况。

### 1.1 技术栈选型
基于现有项目架构，推荐使用以下技术栈：
- **核心框架**: Vue 3 (Composition API) + TypeScript
- **构建工具**: Vite
- **状态管理**: Pinia (用于管理全局数据/WebSocket实时数据)
- **图表库**: **Apache ECharts 5.x** (建议新增依赖，用于复杂图表渲染)
- **UI 组件库**: Element Plus (主要用于弹出层或表单，大屏主体建议手写 CSS/SVG 以保证视觉效果)
- **样式预处理**: SCSS (利用变量和 Mixins 管理主题色)
- **布局适配**: `transform: scale()` 方案或 `vw/vh` 方案 (推荐 Scale 方案以保证像素级还原)
- **动画效果**: CSS3 Animations + GSAP (可选，用于复杂入场动画)

## 2. 总体布局设计
大屏采用经典的 **左-中-右** 三栏布局，整体色调为 **深空蓝/科技黑**。

### 2.1 屏幕分辨率
- **设计基准**: 1920px * 1080px
- **适配策略**: 100% 全屏铺满 (移除 Scale 缩放，采用流式布局)

### 2.2 区域划分
| 区域编号 | 区域名称 | 组件命名建议 | 建议尺寸 |
| :--- | :--- | :--- | :--- |
| **Top** | 顶部标题栏 | `DashboardHeader` | 高度 80px |
| **Left** | 左侧区域 | `LeftPanel` | 高度 calc(100% - 80px), 宽度 25% |
| **Center** | 中间区域 | `CenterPanel` | 高度 calc(100% - 80px), 宽度 50% |
| **Right** | 右侧区域 | `RightPanel` | 高度 calc(100% - 80px), 宽度 25% |

## 3. 详细组件设计

### 3.1 顶部区域 (Header)
- **功能**: 展示系统名称、装饰线条、当前时间。
- **UI元素**:
  - 左侧：Logo、系统名称、**实时时间显示 (YYYY.MM.DD HH:mm:ss)**
  - 中间：装饰性斜角Tab导航 
  - 右侧：时间筛选下拉框（近一月、近一周、近一年）

### 3.2 左侧区域 (Left Column)
垂直排列三个子模块，间距均匀。

#### 3.2.1 左上：服务器总数统计 (Left-Top)
- **标题**: 带渐变背景与发光边框的标题条
- **内容**:
  - 核心指标行：动态旋转的分段发光圆环图标 + 数字 (高亮显示)
  - 分类小卡片 (3个)：带四角光标装饰 (Corner Accents) 的科技感卡片
    - 违规事件总数 (1234)
    - 违规事件总数 (567)
    - 违规事件总数 (4568)

#### 3.2.2 左中：柱状图分析 (Left-Middle)
- **标题**: "这里是标题名称" (e.g., 流量趋势分析)
- **内容**: 垂直柱状图 (Bar Chart)
- **图表细节**:
  - 配色方案: 亮蓝 (#00BFFF) / 亮绿 (#00E09E)
  - 图例样式: 右上角圆形图标 (Circle Icon)
  - 坐标轴: 隐藏刻度线，使用极淡透明网格线，灰色标签字体
  - 悬浮交互: 阴影指示器 (Axis Pointer Shadow)
  - Y轴: 数值 (0-100)
  - 图例: 数据标题 (数据1, 数据2)

#### 3.2.3 左下：列表/状态表格 (Left-Bottom)
- **标题**: "这里是标题名称" (e.g., 实时告警列表)
- **内容**: 滚动表格/列表
- **字段**: 姓名、部门、电话、状态 (已处理/待处理/处理中)
- **样式**: 
  - 奇偶行背景交替
  - 状态列使用胶囊状 Tag (蓝色/黄色/红色)

### 3.3 中间区域 (Center Column)

#### 3.3.1 中上：核心拓扑图 (Center-Top)
- **标题**: 无 (视觉中心)
- **内容**: 3D样式拓扑结构图
- **元素**:
  - 中心节点: 核心服务器/数据中心 (XXXXX)
  - 周围节点: 机构 (机构总数)
  - 连线: 动态流光线条，表示数据传输
  - 悬浮弹窗 (Tooltip): 显示网络攻击、异常访问的详细IP列表
- **技术实现**: 
  - 简化方案: ECharts Graph 插件 + 自定义 SVG 图片/CSS 动画
  - 交互: 鼠标悬停显示详细数据

#### 3.3.2 中下区域 (Center-Bottom)
水平分为左右两部分。

- **中下左 (Center-Bottom-Left)**
  - **标题**: "这里是标题名称" (数据分类占比)
  - **内容**: 环形图 (Donut Chart)
  - **配色**: 绝密(红) / 机密(橙) / 秘密(黄) / 内部(蓝)
  - **细节**: 标签格式 `{b} {c}` (数值加粗高亮)，带引导线。

- **中下右 (Center-Bottom-Right)**
  - **标题**: "这里是标题名称" (访问趋势折线)
  - **内容**: 多重折线图 (Data Trend)
  - **细节**: 
    - 3条数据流线 (蓝/绿/紫)
    - 样式: 折线 (Symbol Circle) + 区域渐变填充 (Area Gradient)
    - 图例: 顶部右侧圆形图标

### 3.4 右侧区域 (Right Column)
垂直排列三个子模块，统一使用 `.panel-title-bar` 渐变标题条。

- **右上 (Top): 设备总览**
  - **核心卡片**: 深蓝渐变背景，发光六边形图标 (CSS Clip-path + Box-Shadow)，大号数字展示。
  - **详情统计**: 4列布局，带右侧分割线。

- **右中 (Middle): 国产化情况**
  - **统计项**: icon + 3D圆柱底座效果 (CSS Radial-gradient)。
  - **进度条**: 
    - 亮绿(#00E09E) -> 亮蓝(#00F0FF) 渐变。
    - 末端悬浮 Tooltip 标签 ("国产化 服务器 112")。

- **右下 (Bottom): 通知列表**
  - **样式**: 垂直列表，深蓝透明背景，亮蓝边框 (#0070C0)。
  - **图标**: 前置发光六边形图标。
  - **交互**: Hover 边框高亮。

- **图片资源**: `src/assets/images/[功能模块]/` (如 `src/assets/images/dashboard/`)

## 5. 颜色规范 (参考)
- **背景色**: `#0B1325` (深蓝底色)
- **主色调**: `#00F0FF` (青色, 高亮文本/边框)
- **辅助色**: 
  - 绿色: `#00E09E`
  - 黄色: `#FFD700` (告警/重点)
  - 蓝色: `#1E90FF` (普通数据)
- **文字色**: `#FFFFFF` (主要), `#A0A0A0` (次要)
